# Spring-Project-PONKU
스프링 + JSP 폰케이스 판매 사이트

## 🖥️ 프로젝트 소개
CASEDIFY 를 참고하여 만든 폰케이스 판매 사이트입니다.
<br>

## 🕰️ 개발 기간
* 24.03.18일 - 24.04.04일 (약 3주)

### 🧑‍🤝‍🧑 맴버구성
정은 - 로그인 / 커스텀 케이스 / 쇼핑카드 / 구매 / 구매내역
수영 - 회원가입 / 마이페이지
범근 - 리뷰 등록 / 모아보기 게시판
성은 - 문의 게시판
나연 - 전체 상품 리스트 / 관리자 모드

### ⚙️ 개발 환경
- `Java 11`
- `JDK 11.0.05`
- **IDE** : STS 3.9
- **Framework** : Spring(3.x)
- **Database** : Oracle DB(11xe)
- **ORM** : Mybatis
